# ai-search-mcp-server
AI搜索的MCP server
可以使用日语/英语/中文进行搜索，日语效果最佳

Claude desktop的配置如下：
"AISearch": {
    "command": "npx",
    "args": ["github:XvJunyan/ai-search-mcp-server"],
    "env": {
        "SEARCH_API_URL": "ws:xxxxxxxxx/xxxxxxxxx",
        "SEARCH_API_KEY": "xxxxxx"
    }
}
